# Plant Disease Detection AI

AI-powered plant disease detection system using TensorFlow, MobileNetV2, and Streamlit.

## Features

- Upload image-based disease prediction
- Live camera disease detection
- Dataset evaluation mode
- Confusion matrix visualization
- Classification report
- Disease information system
- Prevention and treatment recommendations

## Tech Stack

- Python
- TensorFlow
- Keras
- MobileNetV2
- Streamlit
- OpenCV
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
